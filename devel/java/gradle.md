# gradle

<https://gradle.org/>

## Usage

### Daemon

```sh
gradle --status
```

### Info

```sh
gradle projects
gradle tasks
gradle properties
```

### Dependencies

```sh
gradle dependencies
```

### Build

```sh
gradle check
gradle test
gradle build
gradle build -x test
gradle run
gradle clean
gradle publishToMavenLocal -Psigning.password=xxxxxxxx
```
