# gradle

<https://gradle.org/>

## Daemon

```console
$ gradle --status
```

## Info

```console
$ gradle projects
$ gradle tasks
$ gradle properties
```

## Dependencies

```console
$ gradle dependencies
```

## Build

```console
$ gradle check
$ gradle test
$ gradle build
$ gradle build -x test
$ gradle run
$ gradle clean
$ gradle publishToMavenLocal -Psigning.password=xxxxxxxx
```
