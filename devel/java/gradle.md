# gradle

<https://gradle.org/>

Gradle is the open source build system of choice for Java, Android, and Kotlin developers. From mobile apps to microservices, from small startups to big enterprises, it helps teams deliver better software, faster.

## Usage

### Daemon

```sh
gradle --status
```

### Info

```sh
gradle projects
gradle tasks
gradle properties
```

### Dependencies

```sh
gradle dependencies
```

### Build

```sh
gradle check
gradle test
gradle build
gradle build -x test
gradle run
gradle clean
gradle publishToMavenLocal -Psigning.password=xxxxxxxx
```
