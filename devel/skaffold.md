## Install

:::::{tab-set}
::::{tab-item} macOS
:sync: macos

For macOS on Apple silicon:

```console
$ curl -LO https://storage.googleapis.com/skaffold/releases/latest/skaffold-darwin-arm64
$ sudo install skaffold-darwin-arm64 /usr/local/bin/skaffold
```
