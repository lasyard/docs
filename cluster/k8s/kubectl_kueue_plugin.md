# Kubectl Kueue Plugin

## Install

Using krew, see <project:krew.md>.

```console
$ kubectl krew install kueue
Updated the local copy of plugin index.
Installing plugin: kueue
Installed plugin: kueue
\
 | Use this plugin:
 | 	kubectl kueue
 | Documentation:
 | 	https://kueue.sigs.k8s.io/docs/reference/kubectl-kueue/
 | Caveats:
 | \
 |  | Requires the Kueue operator to be installed:
 |  |   https://kueue.sigs.k8s.io/docs/installation/
 | /
/
WARNING: You installed plugin "kueue" from the krew-index plugin repository.
   These plugins are not audited for security by the Krew maintainers.
   Run them at your own risk.
```
