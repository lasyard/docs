# Draw Things

Install in AppStore for MacBook of Apple silicon.

## Models

Put the model files into `~/Library/Containers/com.liuliu.draw-things/Data/Documents/Models`. The location to save the model files can be configured in the model management dialog.

- ERNIE Image Turbo 1.0 [{{hf}}](https://huggingface.co/baidu/ERNIE-Image-Turbo)
- Flux.2 [klein] 4B [{{hf}}](https://huggingface.co/black-forest-labs/FLUX.2-klein-4B)
- Z Image Turbo 1.0 [{{hf}}](https://huggingface.co/Tongyi-MAI/Z-Image-Turbo)
- SD3 Large Turbo 3.5
