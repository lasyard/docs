# jq

<https://jqlang.github.io/jq/>

`jq` is a lightweight and flexible command-line JSON processor.

## Install

::::{plat} centos
:vers: centOS 8.5

```sh
dnf install jq
```

```sh
jq --version
```

{.cli-output}

```text
jq-1.5
```

::::

::::{plat} macos
:vers: macOS Monterey

```sh
brew install jq
```

```sh
jq --version
```

{.cli-output}

```text
jq-1.7.1
```
