# jq

<https://jqlang.github.io/jq/>

## Install

::::{plat} centos
:vers: centOS 8.5

```sh
sudo dnf install jq
```

Check the version:

```console
$ jq --version
jq-1.5
```

::::

::::{plat} macos
:vers: macOS Monterey

```sh
brew install jq
```

Check the version:

```console
$ jq --version
jq-1.7.1
```
