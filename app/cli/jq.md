# jq

<https://jqlang.github.io/jq/>

## Install

::::{tabs}
:::{tab} CentOS 8.5

```console
$ sudo dnf install jq
```

Check the version:

```console
$ jq --version
jq-1.5
```

:::
:::{tab} macOS Monterey

```console
$ brew install jq
```

Check the version:

```console
$ jq --version
jq-1.7.1
```
