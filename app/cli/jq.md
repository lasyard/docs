# jq

<https://jqlang.github.io/jq/>

## Install

::::{tab-set}
:::{tab-item} Ubuntu
:sync: ubuntu

```console
$ sudo apt install jq
```

Check the version:

```console
$ jq --version
jq-1.6
```

:::
:::{tab-item} CentOS 8
:sync: centos8

```console
$ sudo dnf install jq
```

Check the version:

```console
$ jq --version
jq-1.5
```

:::
:::{tab-item} macOS
:sync: macos

```console
$ brew install jq
```

Check the version:

```console
$ jq --version
jq-1.7.1
```

:::
::::
