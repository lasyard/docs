# MediaInfo

<https://github.com/MediaArea/MediaInfo>

MediaInfo is a convenient unified display of the most relevant technical and tag data for video and audio files.

## Install

::::{plat} macos
:vers: macOS Monterey

```sh
brew install mediainfo
```

```sh
mediainfo --version
```

```text
MediaInfo Command line, 
MediaInfoLib - v24.06
```
