# MkDocs

<https://www.mkdocs.org/>

## Install

```console
$ pip install mkdocs
```

```console
$ mkdocs --version
mkdocs, version 1.6.1 from /Users/xxxx/.pyenv/versions/3.12.5/lib/python3.12/site-packages/mkdocs (Python 3.12)
```

## Usage

```console
$ mkdocs new mydoc
INFO    -  Creating project directory: mydoc
INFO    -  Writing config file: mydoc/mkdocs.yml
INFO    -  Writing initial docs: mydoc/docs/index.md
```
