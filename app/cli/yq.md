# yq

<https://github.com/mikefarah/yq>

## Install

:::::{tab-set}
::::{tab-item} macOS
:sync: macos

```console
$ brew install yq
```

::::
::::{tab-item} Ubuntu
:sync: ubuntu

```console
$ curl -LO https://github.com/mikefarah/yq/releases/download/v4.53.3/yq_linux_amd64
$ sudo install yq_linux_amd64 /usr/local/bin/yq
```

::::
:::::

Check the version:

```console
$ yq --version
yq (https://github.com/mikefarah/yq/) version v4.44.3
```
